# /rust:init-dir

Initialize a standard Rust project with Cargo configuration and common setup.

## Instructions

### Step 1: Gather Information

Ask the user:
1. **Project type:**
   - Binary (recommended) - executable application
   - Library - reusable library crate
   - Workspace - multi-crate project

2. **Project name** (used for package name and directory)

3. **Additional features:**
   - Async runtime (tokio/async-std/none)
   - Serialization (serde)
   - CLI parsing (clap)
   - Error handling (thiserror/anyhow)

### Step 2: Initialize with Cargo

**Binary Project:**
```bash
cargo init <project-name>
cd <project-name>
```

**Library Project:**
```bash
cargo init --lib <project-name>
cd <project-name>
```

**Workspace Project:**
```bash
mkdir <project-name>
cd <project-name>
mkdir crates
```

### Step 3: Create Directory Structure

**Binary Project:**
```bash
mkdir -p src tests examples
```

**Library Project:**
```bash
mkdir -p src tests examples benches
```

**Workspace Project:**
```bash
mkdir -p crates/core/src crates/cli/src tests
```

### Step 4: Configure Cargo.toml

For single crate project:

```toml
[package]
name = "<project-name>"
version = "0.1.0"
edition = "2021"
rust-version = "1.75"
description = "A brief description"
license = "MIT OR Apache-2.0"
repository = "https://github.com/username/<project-name>"

[dependencies]

[dev-dependencies]

[profile.release]
lto = true
strip = true
```

For workspace project, create root `Cargo.toml`:

```toml
[workspace]
resolver = "2"
members = ["crates/*"]

[workspace.package]
version = "0.1.0"
edition = "2021"
rust-version = "1.75"
license = "MIT OR Apache-2.0"
repository = "https://github.com/username/<project-name>"

[workspace.dependencies]
# Shared dependencies go here
```

### Step 5: Add Selected Dependencies

Based on user choices, add to `Cargo.toml`:

**Async (tokio):**
```toml
tokio = { version = "1", features = ["full"] }
```

**Serialization:**
```toml
serde = { version = "1", features = ["derive"] }
serde_json = "1"
```

**CLI:**
```toml
clap = { version = "4", features = ["derive"] }
```

**Error handling:**
```toml
thiserror = "2"
anyhow = "1"
```

### Step 6: Create Source Files

**Binary (`src/main.rs`):**
```rust
fn main() {
    println!("Hello, world!");
}
```

**Library (`src/lib.rs`):**
```rust
//! # <project-name>
//!
//! A brief description of the library.

pub fn add(left: u64, right: u64) -> u64 {
    left + right
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn it_works() {
        let result = add(2, 2);
        assert_eq!(result, 4);
    }
}
```

### Step 7: Write .gitignore

Create `.gitignore`:

```gitignore
# Generated by Cargo
/target/
Cargo.lock

# IDE
.idea/
.vscode/
*.swp
*.swo
*~

# OS files
.DS_Store
Thumbs.db

# Environment files
.env
.env.local
*.env

# Build artifacts
*.so
*.dylib
*.dll

# Coverage
*.profraw
*.profdata
coverage/
tarpaulin-report.html
```

Note: Remove `Cargo.lock` from `.gitignore` for binary projects (it should be committed).

### Step 8: Write rustfmt.toml

Create `rustfmt.toml`:

```toml
edition = "2021"
max_width = 100
use_small_heuristics = "Default"
```

### Step 9: Write clippy.toml (optional)

Create `clippy.toml` for custom lint configuration:

```toml
too-many-arguments-threshold = 7
```

### Step 10: Write rust-toolchain.toml (optional)

Create `rust-toolchain.toml` for consistent toolchain:

```toml
[toolchain]
channel = "stable"
components = ["rustfmt", "clippy"]
```

### Step 11: Create Makefile (optional)

Create `Makefile` for common tasks:

```makefile
.PHONY: all build run test lint fmt clean help

all: build

build: ## Build the project
	cargo build

build-release: ## Build release version
	cargo build --release

run: ## Run the application
	cargo run

test: ## Run tests
	cargo test

test-verbose: ## Run tests with output
	cargo test -- --nocapture

lint: ## Run clippy
	cargo clippy -- -D warnings

fmt: ## Format code
	cargo fmt

fmt-check: ## Check formatting
	cargo fmt -- --check

clean: ## Clean build artifacts
	cargo clean

doc: ## Generate documentation
	cargo doc --open

help: ## Show this help
	@grep -E '^[a-zA-Z_-]+:.*?## .*$$' $(MAKEFILE_LIST) | sort | awk 'BEGIN {FS = ":.*?## "}; {printf "\033[36m%-15s\033[0m %s\n", $$1, $$2}'
```

### Step 12: Display Result

Show the created structure using `tree` or `ls -la` and confirm completion.

Run initial commands:
```bash
cargo fmt
cargo clippy
cargo test
```

## Rules

- Use `cargo init` as the starting point
- Follow standard Cargo project layout
- Commit `Cargo.lock` for binaries, optionally for libraries
- Use edition 2021 or later
- Add `rust-version` to specify MSRV
